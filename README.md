# LinyuGameStudio 官网

&gt; 一个极简、单页、免维护的工作室主页，已部署在 GitHub Pages，无需服务器即可在线访问。  
&gt; 在线地址 ➜ [https://linyugameStudio.github.io/linyugamestudio](https://linyugamestudio.github.io/linyugamestudio)

---

## 1. 项目特点
- **零成本**：GitHub Pages 免费托管
- **零依赖**：纯 HTML + CSS + 原生 JS，无构建步骤
- **零维护**：单文件 `index.html`，改完直接推送即生效
- **响应式**：手机 / 平板 / PC 自适应
- **粒子动效**：Canvas 背景，可关可调
- **打字机标题**：循环播放自定义 Slogan
- **一键复制邮箱**：Edge 兼容，不跳 `about:blank`
